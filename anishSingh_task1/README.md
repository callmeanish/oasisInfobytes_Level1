# Responsive Landing Page

A modern and responsive landing page built using **HTML5** and **CSS3** without JavaScript.

## 📌 Objective

Build a visually polished landing page for a fictional technology company while practicing HTML layouts and responsive CSS.

---

## 🚀 Features

- Sticky navigation bar
- Hero section with CTA button
- Features section
- About section
- Responsive design using Flexbox
- Footer with social links
- Consistent color palette
- Mobile-friendly layout
- Clean typography

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)

---

## 📂 Project Structure

```
landing-page/
│── index.html
│── style.css
│── README.md
```

---

## 💻 How to Run

1. Download or clone this repository.

```
git clone https://github.com/your-username/landing-page.git
```

2. Open the project folder.

3. Double-click `index.html`

OR

Open it using **VS Code** and run **Live Server**.

## 📱 Responsive Design

The landing page adapts to:

- Desktop
- Tablet
- Mobile devices

using CSS Flexbox and media queries.
